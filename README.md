# Système de Gestion des Réservations  
Projet en langage C.

## À propos  
Ce projet implémente un système de gestion des réservations écrit en C.  
Il permet :  
- la création, lecture, modification et suppression des réservations,  
- la génération de factures,  
- la gestion simple de fichiers comme stockage persistant,  
- une interface utilisateur en terminal (CLI).

---

## Prérequis  
- Un compilateur C (GCC, Clang, MinGW, etc.).  
- Un OS compatible : Linux, macOS ou Windows (ou WSL).  
- Un terminal ou invite de commande.
